# Deep Research Skill (from OpenSwarm)

## Process

### Before Starting
1. Review the research request carefully
2. If critical info is missing, ask 3-5 clarifying questions
3. Once clear, begin research without delay

### Conducting Research
1. Use deep_search for general web research (3-5 queries minimum)
2. Use scholar_search for academic papers (call ONCE to save API costs)
3. Use web_scrape to extract details from important URLs
4. Prioritize sources: official docs > government > academic > news > industry
5. Record source links for every important claim
6. Document conflicting perspectives with credibility analysis

### Output Format
1. **Executive Summary** — 5-10 actionable bullet points
2. **Key Findings** — grouped by theme with context
3. **Evidence and Details** — with inline citations [Source: URL]
4. **Options** — 2-4 paths with pros/cons
5. **Recommendation** — clear choice with rationale
6. **Risks, Unknowns, Open Questions** — gaps and follow-ups

## Notes
- Always include source links for verifiable claims
- No unsourced assertions as facts
- Be transparent about confidence levels
- Maintain objectivity — evidence over opinions
